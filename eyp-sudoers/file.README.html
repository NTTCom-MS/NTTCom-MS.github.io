Overview

Manage sudoers

Module Description

This module needs /etc/sudoers.d support, which is true for: * RedHat 6 and up * Ubuntu 10.04 and up.

Setup

What sudoers affects

  • Unless overwrite_sudoers is false, /etc/sudoers

  • Creates / deletes files on /etc/sudoers.d

Setup Requirements

This module requires pluginsync enabled

Beginning with sudoers

basic example:

class { 'sudoers': }

sudoers::sudo { 'vagrant':
  withoutpassword => true,
}

Usage

Add users with full sudo access:

sudos:
  adminuser:
    withoutpassword: true
  cpiscina: {}
  mlleidebrad: {}
  mtelevisio: {}

Restrict sudo to a specific command:

sudos:
  ppt-deploy:
    command: /etc/init.d/nginx
  ppt-deploy-service:
    username: bbt-deploy
    command: /usr/bin/service

sudoers::defaults:

sudoers::defaults { '!requiretty':
  username => 'nrpe',
}
